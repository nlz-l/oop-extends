# Java 继承练习模块

这是一个用于 Java 面向对象继承练习的 IntelliJ IDEA 模块目录，目前主要保留项目元数据文件。实际的继承练习代码更集中地放在 `java-learning-notes/oop-extends` 中。

## 当前内容

```text
oop-extends.iml    IntelliJ IDEA 模块配置文件
.idea/             本地 IDE 配置目录
```

## 适用场景

- 作为 IntelliJ IDEA 中单独导入的练习模块
- 保存面向对象继承练习的项目结构
- 与 `java-learning-notes` 中的继承代码互相对照

## 建议查看

如果需要阅读完整的继承学习代码，建议优先查看：

```text
java-learning-notes/oop-extends
```

该目录通常包含父类、子类、方法重写、构造方法调用、访问权限等更完整的练习内容。
